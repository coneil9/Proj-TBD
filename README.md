## Gradient Composer
 is a fully interactive web application that lets users create custom CSS gradients through an intuitive, real-time visual interface. Built with React, TypeScript, and Tailwind CSS, the app allows users to add and adjust color stops, modify gradient types, customize angles, and instantly export clean CSS code.

Designed to be lightweight, elegant, and user-friendly, this tool is perfect for designers, developers, and anyone who wants to generate polished gradients quickly. Also, a simple project allowing me to practice my front-end development. 

# Features
Gradient Customization

Choose linear or radial gradients
Adjust angle for linear gradients (0–360°)
Add, edit, and delete color stops
Control each stop’s position using sliders (0–100%)
Save favorite gradients as presets
Load previous gradients from LocalStorage
Share gradients via URL parameters

# Live Preview

Real-time visual preview of the gradient
Checkerboard background for accurate contrast
Responsive layout for all devices

# CSS Export

Auto-generated CSS gradient string
One-click copy to clipboard
Clean, formatted output ready for production
